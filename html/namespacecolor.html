◆ colorize()

template<typename T>
std::string color::colorize ( const std::string & color,
const T & text )

Builds and returns a colored string without printing it.

This function wraps the given text with ANSI color codes and returns it as a std::string. It is useful for preparing formatted output to be printed later.

Template Parameters
TThe type of the text (e.g., std::string, int, float, etc.).
Parameters
colorThe ANSI color escape code to use.
textThe content to wrap with color.
Returns
std::string The formatted colored string with reset code.
Note
The string is not printed to the terminal — it is returned to the caller.
